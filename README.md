# Grok's AI Superhero Adventure 🚀🦸‍♂️

**Interactive educational web game for kids (age 8-12)** teaching Gitlawb, OpenClaude, OpenGateway, Grok & xAI through fun space missions.

## Features
- Choose-your-own-adventure style game
- Educational missions on AI tools & connectors
- Score tracking with localStorage
- Responsive design, mobile-friendly
- Shareable results
- MIT License - fork & remix!

## How to Play
1. Open `index.html` in browser
2. Team up with Grok, OpenClaude etc.
3. Complete missions, learn real concepts

## Tech Stack
- Pure HTML/CSS/JS (no build needed)
- Local storage for progress
